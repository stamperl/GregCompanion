Click Foundry release notes

r0.2.0 LV Power and Automation

A progression release focused on making the jump from Steam Age into LV feel like a real factory upgrade: configured fluids, stronger power planning, tougher machine parts, and a refreshed quest book.

Coke oven control

The Coke Oven is now a real multiblock flow problem, with wrench-selected liquid output and clearer creosote handling.

LV power grid

LV now has 1A, 2A, 4A, and 8A cable and buffer progression, with the Arc Furnace demanding a serious 4A supply.

Guided progression

The quest book now teaches the new chains instead of leaving players to guess why batteries, motors, and machine parts matter.

Save Upgrade Notes

  • Players with old single-block Coke Ovens should replace them with the four Coke Oven Blocks granted by the migration prompt.
  • Existing LV setups may need cable and battery buffer upgrades before high-draw machines will run reliably.
  • Lead and sodium salt now require a diamond pickaxe path, so battery progression has a clearer mining gate.

Coke Oven and Fluids

  • The Coke Oven now uses a configured output face for creosote rather than automatically draining from any convenient side.
  • Wrenching the multiblock opens a shared output configuration page for the full structure.
  • Factory floor graphics now show the selected liquid output direction so the route is visible without reopening every panel.
  • Creosote output per smelt has been reduced, while Coke Oven processing speed has been increased to make it feel more active without flooding early storage.
  • Tanks and pipes now present their current contents generically, so the UI is ready for more fluid types later.

LV Power and Batteries

  • LV power now has amperage tiers: 1A, 2A, 4A, and 8A tin cables.
  • Battery buffers now follow the same amp tiers, and each buffer slot represents a real battery installed in the machine.
  • The Arc Furnace now expects a 4A LV feed, making four turbines and proper stored power a meaningful target.
  • Battery crafting has been rebuilt around battery alloy, empty cells, sodium batteries, and higher-capacity lithium batteries.
  • Lithium now comes from LV electrolyzer processing instead of a simple mined resource.

LV Machine Backbone

  • New LV backbone parts have been added: motors, pistons, pumps, conveyors, steel gears, steel rings, and steel screws.
  • LV machine recipes now lean on mechanical parts and tool-shaped crafting instead of overusing circuits.
  • The LV Assembler now supports a more appropriate machine interface with six item inputs and one liquid slot.
  • The LV Canner now has a proper role in filling battery cells with sodium or lithium chemistry.
  • Machine and resource icons for the recent LV additions have been regenerated and cleaned for the factory floor and recipe browser.

Storage, Hoppers, and Routing

  • Standard Chests and Hoppers introduce early factory-floor item storage and basic machine feeding.
  • Hoppers are configured with the wrench and push in the selected direction, matching the pipe-style setup language.
  • Pipes now have internal buffers and show current contents and flow rate when inspected.
  • Cables no longer use fluid-style flow directions; LV machines draw from nearby power based on cable capacity and demand.

Quest Book Refresh

  • The Steam Age guide now calls out factory tools, chest and hopper automation, and deliberate fluid routing.
  • The Coke and Steel line now explains the 2x2 Coke Oven, creosote drain setup, BBF staging, and why steel is the end goal of the Steam Age.
  • The LV line now walks through red alloy wiring, rubber, vacuum tubes, wood pulp boards, the first circuit, turbine power, steel mechanical parts, motors, motion parts, batteries, amp cables, and aluminium prep.
  • The Multiblocks page remains focused on the larger structures, while the main age lines explain why each structure matters to progression.

Balance Direction

  • Steam machines remain useful as repeatable automation, while LV machines are now more expensive but more efficient where it matters.
  • Battery and cable choices are now part of planning a factory instead of being decorative upgrades.
  • The Arc Furnace is intentionally a heavy LV milestone, not just another machine to place once recipes are unlocked.
